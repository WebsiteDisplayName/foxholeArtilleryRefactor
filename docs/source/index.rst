Foxhole Artillery Refactor Documentation
===================================

**Foxhole Artillery Refactor** is a Python desktop app that performs artillery calculations
for `Foxhole <https://store.steampowered.com/app/505460/Foxhole/>`_.

It is recommended that the program be overlayed over the game using 
`OnTopReplica <https://github.com/LorenzCK/OnTopReplica>`_.

Refer to the :doc:`usage` section for functionality and :doc:`installation` for installing
the project.

.. note::

   This project is under active development.
